---
description: Sincronizar alterações com GitHub e Vercel
---

Este workflow automatiza o envio de todas as alterações locais para o GitHub e, consequentemente, para a Vercel.

// turbo-all
1. Adicionar todas as alterações
   `git add .`
2. Criar um commit com as mudanças
   `git commit -m "Update: Refinements and removals"`
3. Enviar para o GitHub
   `git push origin main`
